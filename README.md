## Install
and  provision an NGINX server in less than a minute using Docker on Mac:

`open -a Docker`

`mkdir learn-terraform-docker-container`

`cd learn-terraform-docker-container`

`vi main.tf`

`!wq`

`terraform init`

`terraform apply`

`terraformyes destroy`
